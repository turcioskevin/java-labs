# 🎵 PlayListManager

**PlayListManager** is a simple Java program that reads a list of songs from a text file and checks for duplicates in your playlist.  
It prints out any songs that appear more than once, helping you manage and clean your playlist.

---

## 📌 Features

- Reads a playlist from a text file (`PlayList.txt`)  
- Detects duplicate songs  
- Prints duplicate songs to the console  
- Uses only explicit imports (no `import java.util.*`)  

---

## 🛠️ Technologies Used

- Java (JDK 8 or higher)  
- Core Java libraries: `java.util.ArrayList`, `java.util.Scanner`, `java.io.FileInputStream`, `java.io.IOException`  

---

## 🚀 Getting Started

### Prerequisites

- Java installed (JDK 8+)  
- `PlayList.txt` file in the same directory as the program  

### ▶️ How to Run

1. Save your playlist in a text file named `PlayList.txt`.  
   - Each song should be on a separate line.  

2. Compile the program:

```bash
javac PlayListManager.java
