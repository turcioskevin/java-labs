# Treasure Finder

## Description
This program searches a 2D grid to locate a hidden treasure marked by a '+' symbol.

## Features
- Traverses a 2D array using nested loops
- Identifies a specific target character
- Outputs the location using row/column coordinates

## Concepts Used
- 2D Arrays
- Nested loops
- Conditional statements

## Example Output
Treasure found at row 9, column 1

## How to Run
javac TreasureFinder.java
java TreasureFinder
