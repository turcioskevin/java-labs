# 🌡️ Temperature Converter (Java Swing)

A simple **Temperature Converter** application built in Java using **Swing**. This GUI-based app allows users to convert temperatures between **Fahrenheit** and **Celsius** quickly and easily.  

---

## 📌 Features

- Convert **Fahrenheit → Celsius** and **Celsius → Fahrenheit**  
- Input fields for both units  
- Real-time detection of which field was edited last  
- Simple and clean GUI  
- Error handling for invalid input  

---

## 🛠️ Technologies Used

- Java  
- Swing (GUI framework)  
- AWT (event handling and layout)  

---

## 🚀 Getting Started

### Prerequisites

- Java JDK 8 or higher installed  
- Any Java IDE (IntelliJ, Eclipse, VS Code) or terminal
